# 市场相关数据源

## GitHub热门项目

### yfinance
- 描述: 从Yahoo! Finance API下载市场数据
- GitHub stars: 22,327
- 语言: Python
- 许可证: Apache License 2.0
- 地址: https://github.com/ranaroussi/yfinance

## 其他可能的市场数据源

1. Alpha Vantage - 提供免费股票市场数据API
2. IEX Cloud - 提供实时股票市场数据
3. Finnhub - 提供实时市场数据和新闻
4. Polygon.io - 提供股票、外汇、加密货币市场数据
5. Trading Economics - 提供宏观经济数据

## 注意事项
- 免费API通常有速率限制
- 商业用途可能需要付费订阅
- 部分API需要注册获取API密钥
